Assignments

Assignment 1 - Basic Html

This assignment requires creating a structured website with a main page that serves as a portal to all course assignments, following a specific directory structure. Assignment 1 involves designing a standalone webpage on a chosen topic with proper HTML structure, formatted content, images, a list, a table, and external links while ensuring originality and proper documentation.

Assignment 2 - Basic CSS

This assignment involves enhancing the main page by adding styles such as a custom font, color scheme, background, formatted div, and styled links while ensuring a cohesive theme. Assignment 2 requires creating a new webpage on a chosen topic, including structured content with navigation links, images, a table, and well-formatted styles using element, ID, and class selectors to improve readability and design consistency.

Assignmnet 3 - Page Layout

This assignment focuses on enhancing the design of the main portal page by structuring it with a traditional HTML layout using header, content, and footer sections while styling it with Flexbox and CSS. It requires incorporating assignment screenshots, unique descriptions, responsive design with media queries, and ensuring a visually appealing layout that matches the provided diagram.

Assignment 4 - Recreate CSS Page

This assignment requires recreating a specified webpage using HTML and CSS while ensuring structural accuracy and matching colors with the original design. It focuses on flexbox layout skills, simplified navigation for small screens, placeholder images, and Lorem Ipsum text, while omitting advanced effects like parallax, animations, and the search box.
